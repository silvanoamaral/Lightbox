# Lightbox

## Project Structure

```
├── index.html
├── src
│   ├── js
│   │   └── lightbox.js
│   │   └── content.js
│   ├── css
│       └── lightbox.css
├── README.md

```

* `index.html` file main
* `src/js/lightbox.js` contém código para iniciar carousel
* `src/js/content.js` contém código que faz a requisição no endpoint e preenche o lightbox 
* `src/css/lightbox.css` contém todo css
